MongoDB常用命令

MongoDB 此备忘单包含一些方便的提示、命令和快速参考，可让您立即连接并进行 CRUD

连接到 MongoDB

  • mongo 命令用于连接到 MongoDB。
  • 如果未指定参数，则默认连接到 mongodb://127.0.0.1:27017。
  • --host 参数用于指定主机名或 IP 地址。
  • --port 参数用于指定端口号，默认为 27017。
  • -u 参数用于指定用户名。
  • -p 参数用于指定密码，如果需要提示，则省略密码。
  • URI 连接格式为：mongodb://<username>:<password>@<host>:<port>/<dbname>。

MongoDB Atlas

  • mongo 命令用于连接到 MongoDB。
  • mongodb+srv:// URI 连接格式用于连接 MongoDB Atlas。
  • <cluster-name> 为 Atlas 群集名称。
  • <username> 为 Atlas 帐户用户名。
  • <dbname> 为要连接的数据库名称。

显示数据库

  • show dbs 命令用于显示当前 MongoDB 实例中所有的数据库。

显示当前数据库

  • db 命令用于显示当前数据库的名称。

MongoDB Index Options

  • createIndex() 方法用于创建索引。
  • {"lastModifiedDate": 1} 表示以 lastModifiedDate 键为升序创建单字段索引。
  • {expireAfterSeconds: 3600} 参数指定文档将在 3600 秒后过期。
  • createIndex() 方法用于创建索引。
  • {"name": 1} 表示以 name 键为升序创建单字段索引。
  • {unique: true} 参数指定索引中不允许重复值。
  • createIndex() 方法用于创建索引。
  • {"name": 1} 表示以 name 键为升序创建单字段索引。
  • {partialFilterExpression: {age: {$gt: 18}}} 参数指定创建部分索引，只索引 age 大于 18 的文档。
  • createIndex() 方法用于创建索引。
  • {"name": 1} 表示以 name 键为升序创建单字段索引。
  • {collation: {locale: 'en', strength: 1}} 参数指定创建区分大小写或不区分大小写的索引，具体取决于 strength 的值。在此示例中，强度为 1 表示不区分大小写。
  • createIndex() 方法用于创建索引。
  • {"name": 1} 表示以 name 键为升序创建单字段索引。
  • {sparse: true} 参数指定创建稀疏索引，仅索引包含 name 键的文档，而不索引缺少 name 键的文档。

列表索引

  • getIndexes() 方法用于获取一个集合中的所有索引。
  • getIndexKeys() 方法用于获取一个集合中的所有索引的键。

删除索引

  • dropIndex() 方法用于删除指定索引。

隐藏/取消隐藏索引

  • hideIndex() 方法用于将指定索引隐藏，以防止被使用。在后续的操作中，隐藏的索引将不会被考虑。
  • unhideIndex() 方法用于取消隐藏指定索引。这使得该索引能够再次被使用。
