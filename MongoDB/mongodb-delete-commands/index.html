MongoDB常用CURD命令之删除

发表于2021-04-16|MongoDB
|阅读量:

删除文档的命令

  • remove() 方法用于删除一个或多个匹配的文档。
  • {name: "zhangsan"} 参数表示筛选条件，删除所有 name 值为 “zhangsan” 的文档。
  • {justOne: true} 参数表示删除匹配到的第一个文档。
  • 警告！删除所有文档但不删除集合本身及其索引定义。
  • {name: "zhangsan"} 参数表示筛选条件，删除所有 name 值为 “zhangsan” 的文档。
  • "writeConcern": {"w": "majority", "wtimeout": 5000} 参数指定写关注选项。
  • findOneAndDelete() 方法查找并删除单个文档。
  • {name: "zhangsan"} 参数表示筛选条件，删除 name 值为 “zhangsan” 的第一个文档。

检查主节点状态

  • isMaster() 方法用于检查当前节点是否为主节点，并返回一个包含各种节点信息的文档。

获取主机信息

  • hostInfo() 方法用于返回当前 MongoDB 实例的主机信息。

打印分片状态

  • printShardingStatus() 方法用于打印分片集群的状态信息。

关闭服务器

  • shutdownServer() 方法用于关闭当前 MongoDB 服务器。

设置允许从节点查询

  • setSlaveOk() 方法用于设置当前连接允许从节点查询。

获取从节点查询状态

  • getSlaveOk() 方法用于获取当前连接是否允许从节点查询。

获取查询分析等级

  • getProfilingLevel() 方法用于获取当前的查询分析等级，0 表示关闭查询分析，1 表示开启查询分析。

获取查询分析状态

  • getProfilingStatus() 方法用于获取查询分析的状态，返回一个包含查询分析等级和查询分析器输出的文档。

设置查询分析等级

  • setProfilingLevel() 方法用于设置查询分析等级和慢查询阈值。第一个参数表示查询分析等级，第二个参数表示慢查询阈值（毫秒），0 表示关闭查询分析。

启用监视

  • enableFreeMonitoring() 方法用于启用 MongoDB 的免费监视服务。

禁用监视

  • disableFreeMonitoring() 方法用于禁用 MongoDB 的免费监视服务。

获取监视状态

  • getFreeMonitoringStatus() 方法用于获取 MongoDB 的免费监视服务状态信息。

创建视图

  • createView() 方法用于在当前数据库中创建一个视图。第一个参数为视图名称，第二个参数为视图的源集合名称，第三个参数为一个操作数组，用于指定视图的操作。在上述示例中，我们使用 $project 操作符来仅保留文档中的 department 键。
