MongoDB常用CURD命令之查找

发表于2021-04-14|MongoDB
|阅读量:

寻找文件

  • db.document.findOne():

    该命令用于查找一个随机文档。在集合 document 中执行该命令，会返回一个该集合中任意一个文档。

  • db.document.find().prettyPrint():

    该命令用于查找所有文档，并以格式化的形式打印输出。在集合 document 中执行该命令，会返回该集合中所有文档并将其格式化。

  • db.document.find({}, {name:true, _id:false}):

    该命令用于仅显示文档 document 的名称。在集合 document 中执行该命令，会返回该集合中所有文档的名称，并且不包含默认的 _id 属性。

  • db.document.find({属性名: 属性值}, {name:true, _id:false}):

    该命令可以在多个文档中按属性查找一个或多个文档。在集合 document 中执行该命令，会返回该集合中所有包含指定属性值的文档，并且不包含默认的 _id 属性。可以根据需要修改 属性名 和 属性值，以匹配想要查找的文档。

使用运算符查找文档

大于 ‘$gt’

该命令用于查找 class 属性的值大于指定值 'T' 的所有文档。

大于等于 ‘$gte’

该命令用于查找 class 属性的值大于或等于指定值 'T' 的所有文档。

小于 ‘$lt’

该命令用于查找 class 属性的值小于指定值 'T' 的所有文档。

小于等于 ‘$lte’

该命令用于查找 class 属性的值小于或等于指定值 'T' 的所有文档。

属性是否存在 ‘$exists’

该命令用于查找包含指定属性的所有文档。

正则表达式匹配’$regex’

按元素类型搜索’$type’

查找

以下是MongoDB命令的用法和参数作用的修改版：

返回单个文档的命令

  • findOne() 方法用于返回符合筛选条件的单个文档。
  • filter 参数指定筛选条件，可以为空表示返回集合中的任何一个文档。
  • projection 参数指定要返回的字段，可以为空表示返回文档的所有字段。

返回多个文档的命令

  • find() 方法用于返回符合筛选条件的多个文档。
  • filter 参数指定筛选条件，可以为空表示返回集合中的所有文档。
  • projection 参数指定要返回的字段，可以为空表示返回文档的所有字段。
  • limit 参数指定返回文档的数量上限，默认值为20。
  • skip 参数指定从哪个文档开始返回，用于分页，默认值为0。
  • sort 参数指定文档的排序方式，可以是升序（1）或降序（-1），默认值为按照_id升序排列。

显示美化后的多个文档的命令

  • pretty() 方法用于美化输出，使返回的文档更易读。

带筛选条件的多个文档的命令

  • filter 参数指定筛选条件，可以是单个条件，也可以是多个条件的逻辑与（AND）。

带ISO日期类型的筛选条件的多个文档的命令

  • 用ISO日期格式的值作为筛选条件时需要使用ISODate()函数进行转换。

查询执行计划的多个文档的命令

  • explain() 方法可以用于查询MongoDB的查询执行计划，executionStats 参数表示返回详细的执行统计信息。

返回集合中不同字段值的命令

  • distinct() 方法用于返回集合中某个字段的不同值。
  • field 参数指定要查询不同值的字段。
  • filter 参数指定筛选条件，可以为空表示返回集合中所有文档的指定字段的不同值。

以下是修改版：

计算文档数量的命令

  • countDocuments() 方法用于计算符合筛选条件的文档数量。
  • filter 参数指定筛选条件，可以为空表示计算集合中所有文档的数量。
  • estimatedDocumentCount() 方法用于基于元数据估计集合中的文档数量，返回的是一个近似值。
  • count() 方法用于计算符合筛选条件的文档数量，基于元数据估计结果。

Comparison 比较

  • $gt 表示大于（greater than）操作符，用于返回指定字段大于给定值的文档。
  • $gte 表示大于等于（greater than or equal to）操作符，用于返回指定字段大于等于给定值的文档。
  • $lt 表示小于（less than）操作符，用于返回指定字段小于给定值的文档。
  • $lte 表示小于等于（less than or equal to）操作符，用于返回指定字段小于等于给定值的文档。
  • $ne 表示不等于（not equal to）操作符，用于返回指定字段不等于给定值的文档。
  • $in 表示在给定数组中（in）操作符，用于返回指定字段值在给定数组中的文档。
  • $nin 表示不在给定数组中（not in）操作符，用于返回指定字段值不在给定数组中的文档。

计算文档数量的命令

  • countDocuments() 方法用于计算符合筛选条件的文档数量。
  • filter 参数指定筛选条件，可以为空表示计算集合中所有文档的数量。
  • estimatedDocumentCount() 方法用于基于元数据估计集合中的文档数量，返回的是一个近似值。
  • count() 方法用于计算符合筛选条件的文档数量，基于元数据估计结果。

Comparison 比较

  • $gt 表示大于（greater than）操作符，用于返回指定字段大于给定值的文档。
  • $gte 表示大于等于（greater than or equal to）操作符，用于返回指定字段大于等于给定值的文档。
  • $lt 表示小于（less than）操作符，用于返回指定字段小于给定值的文档。
  • $lte 表示小于等于（less than or equal to）操作符，用于返回指定字段小于等于给定值的文档。
  • $ne 表示不等于（not equal to）操作符，用于返回指定字段不等于给定值的文档。
  • $in 表示在给定数组中（in）操作符，用于返回指定字段值在给定数组中的文档。
  • $nin 表示不在给定数组中（not in）操作符，用于返回指定字段值不在给定数组中的文档。

Element 元素查询

  • {name: {$exists: true}} 查询所有具有“name”键的文档。
  • {"zipCode": {$type: 2}} 查询所有具有“zipCode”键且值为字符串的文档。
  • {"zipCode": {$type: "string"}} 查询所有具有“zipCode”键且值为字符串的文档。

Aggregation Pipeline 聚合管道

  • aggregate() 方法可用于对集合执行聚合操作，返回一个文档数组。
  • $match 管道用于筛选符合条件的文档。
  • $group 管道用于分组和聚合文档。
  • $sum 操作符可用于求和。
  • $sort 管道可用于排序聚合结果。

使用“文本”索引进行文本搜索

  • $text 索引可用于在文本字段上执行全文搜索。
  • { $search: "cake" } 表示搜索包含单词“cake”的文本字段。
  • { score: { $meta: "textScore" } } 用于将文档的搜索得分作为结果的“score”键返回。
  • sort() 方法用于对结果进行排序。

Regex 正则表达式查询

  • {name: /^zhangsan/} 查询所有以“zhangsan”开头的文档。
  • {name: /^zhangsan$/i} 查询所有名称为“zhangsan”的文档，不区分大小写。

Array

  • find() 方法用于在集合中查找匹配的文档。
  • {tags: {$all: ["Realm", "Charts"]}} 参数表示查找所有包含 Realm 和 Charts 标签的文档。
  • find() 方法用于在集合中查找匹配的文档。
  • {field: {$size: 2}} 参数表示查找所有具有两个元素的 field 数组的文档。
  • find() 方法用于在集合中查找匹配的文档。
  • {results: {$elemMatch: {product: "xyz", score: {$gte: 8}}}} 参数表示查找所有包含 results 数组，且该数组至少包含一个元素，满足 product 键等于 "xyz"，score 键大于或等于 8 的文档。

Projections

  • find() 方法用于在集合中查找匹配的文档。
  • {"x": 1}, {"actors": 1} 参数表示只返回所有 x 等于 1 的文档中的 actors 数组和 _id 键。
  • find() 方法用于在集合中查找匹配的文档。
  • {"x": 1}, {"actors": 1, "_id": 0} 参数表示只返回所有 x 等于 1 的文档中的 actors 数组，但不返回 _id 键。
  • find() 方法用于在集合中查找匹配的文档。
  • {"x": 1}, {"actors": 0, "summary": 0} 参数表示只返回所有 x 等于 1 的文档中除了 actors 和 summary 键之外的所有内容。

Sort, Skip, Limit

  • find() 方法用于在集合中查找匹配的文档。
  • {} 参数表示查找所有文档。
  • sort({"year": 1, "rating": -1}) 参数表示按照 year 键进行升序排序，然后按照 rating 键进行降序排序。
  • skip(10) 参数表示跳过前10个匹配的文档。
  • limit(3) 参数表示返回匹配的文档中的前3个文档。
