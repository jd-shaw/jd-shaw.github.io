MongoDB 不常用其他命令

命令

创建用户

  • use admin 命令用于切换到 admin 数据库。
  • db.createUser() 命令用于创建新用户，其中 user 参数指定用户名，pwd 参数指定密码，roles 参数指定用户的角色。本例中创建的用户拥有 root 角色，即具有最高权限。

删除用户

  • db.dropUser() 命令用于删除用户。本例中删除的是用户名为 root 的用户。

验证用户身份

  • db.auth() 命令用于验证用户身份，其中 user 参数指定用户名，passwordPrompt() 函数用于输入密码。

切换数据库

  • use 命令用于切换数据库。本例中切换到名为 test 的数据库。

获取兄弟数据库

  • db.getSiblingDB() 命令用于获取兄弟数据库。本例中获取的是名为 dbname 的兄弟数据库。

查看当前操作

  • db.currentOp() 命令用于查看当前操作。

终止操作

  • db.killOp() 命令用于终止指定的操作，其中 123 为操作的 opid。

文件锁定与解锁

  • db.fsyncLock() 命令用于锁定文件系统，以便进行备份等操作。
  • db.fsyncUnlock() 命令用于解锁文件系统。

获取集合列表

  • db.getCollectionNames() 命令用于获取当前数据库中的所有集合列表。

获取集合信息

  • db.getCollectionInfos() 命令用于获取当前数据库中所有集合的详细信息。

打印集合状态

  • db.printCollectionStats() 命令用于打印当前集合的状态信息。

打印数据库状态

  • db.stats() 命令用于打印当前数据库的状态信息。

获取复制信息

  • db.getReplicationInfo() 命令用于获取当前副本集的复制信息。

打印复制信息

  • db.printReplicationInfo() 命令用于打印当前副本集的复制信息。

监控集合变化的命令

  • watch() 方法用于监控指定集合的变化。
  • $match 用于筛选变化类型，此处为 "operationType": "insert"，表示只监控插入操作。
  • while 循环用于检查变化是否被处理完，isExhausted() 方法用于检查光标是否已被耗尽。
  • hasNext() 方法用于检查光标是否有下一个文档，next() 方法用于返回光标的下一个文档。

副本集

查看副本集状态

  • rs.status() 命令用于查看副本集的状态。该命令将返回有关每个成员及其健康状态、数据同步状态和副本集配置的信息。

初始化副本集

  • rs.initiate() 命令用于初始化一个新的副本集。该命令将创建一个新的副本集配置，并将当前节点作为副本集的主节点。
  • _id 参数指定副本集的唯一标识符。
  • members 参数指定副本集中的成员列表，每个成员由其在配置中的唯一 _id 和主机名 host 组成。 arbiterOnly 参数指定一个仲裁节点。

添加副本集成员

  • rs.add() 命令用于将一个新的节点添加到副本集中。该命令将返回一个包含添加操作结果的对象。
  • 参数是要添加到副本集中的新节点的主机名和端口号。

添加仲裁节点

  • rs.addArb() 命令用于将一个新的仲裁节点添加到副本集中。该命令将返回一个包含添加操作结果的对象。
  • 参数是要添加到副本集中的新仲裁节点的主机名和端口号。

从副本集中删除成员

  • rs.remove() 命令用于从副本集中删除一个成员。该命令将返回一个包含删除操作结果的对象。
  • 参数是要从副本集中删除的节点的主机名和端口号。

查看副本集配置

  • rs.conf() 命令用于返回当前副本集的配置。

查看副本集主节点状态

  • rs.isMaster() 命令用于返回副本集中当前的主节点。该命令将返回一个包含当前主节点信息的对象。

查看副本集复制信息

  • rs.printReplicationInfo() 命令用于查看有关副本集复制状态和性能的信息。

重新配置副本集

  • rs.reconfig() 方法用于重新配置副本集。
  • <valid_conf> 参数为一个 JSON 对象，包含副本集配置信息。

允许从节点读取数据

  • rs.slaveOk() 方法用于允许从节点读取数据。

降级副本集主节点

  • rs.stepDown() 方法用于降级当前副本集主节点，使其成为从节点。
  • 20 表示新的主节点选举超时时间（秒）。
  • 5 表示等待新的主节点选举完成的时间（秒）。
